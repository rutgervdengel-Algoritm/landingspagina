# WachtlijstHelderheid - Landingspagina

## Wat is dit project?

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Taal**: TypeScript
- **Styling**: Tailwind CSS met custom design tokens
- **Animaties**: Framer Motion (`motion`, `whileInView`)
- **Icons**: Lucide React
- **Font**: Plus Jakarta Sans (Google Fonts)
- **Deploy**: Vercel
